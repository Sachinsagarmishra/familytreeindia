# Family Tree India 🌳

A modern, high-performance landing page for Family Tree India, focused on community-driven reforestation and student-led tree guardianship.

## 🚀 Key Features

- **Modern Aesthetic:** A premium design featuring dark/light mode sections, elegant typography (Fraunces & Instrument Sans), and high-quality imagery.
- **Interactive Transformation Slider:** A custom-built environmental progress slider that visualizes the impact of plantation drives over time.
- **Infinity Loop Marquee:** A smooth, infinite-scrolling partner logo strip.
- **Fully Responsive:** Optimized for all devices with tailored font scaling and layout shifts.
- **Clean URLs:** Integrated `.htaccess` for professional URL structures (removing `.html` extensions).
- **Performance Optimized:** Modularized CSS and JavaScript for faster load times and better maintainability.

## 🛠 Tech Stack

- **HTML5:** Semantic structure.
- **Vanilla CSS:** Custom design system with modern properties like `clamp()`, `mask`, and CSS variables.
- **Vanilla JavaScript:** Custom reveal animations, cursor tracking, and interactive slider logic.
- **Apache:** Clean URL routing via `.htaccess`.

## 📁 Project Structure

```text
├── index.html       # Main structure
├── style.css        # Core design system & layouts
├── script.js        # Interactive logic & animations
├── .htaccess        # URL routing rules
├── Icons/           # Partner logos
└── img/             # High-res visual assets
```

## 🔧 Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Sachinsagarmishra/familytreeindia.git
   ```
2. Open `index.html` in your browser, or host on an Apache server to enable the `.htaccess` clean URL features.
